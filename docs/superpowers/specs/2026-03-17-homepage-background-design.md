# Homepage Background And Mobile Landing Design Spec

## Overview

为《打工之道》首页增加一张玩家向主视觉背景图，并同步将首页适配移动端使用。目标是让首页既能在桌面端建立“深夜办公室里的打工人生”氛围，也能在手机上保持清晰、可操作，并允许用户直接从手机首页进入游戏。

本次设计的重点是“首页”而不是“整个游戏”。也就是说，我们会一起设计：

- 一张服务首页的主视觉背景图
- 桌面端首页如何使用这张图
- 移动端首页如何在保留氛围的前提下实现可用性优先

本轮不包含整套游戏界面的完整移动端重构。

## Design Goal

- 第一眼就让玩家感受到“这是一个 AI 驱动的职场模拟器”
- 强化“深夜、办公室、打工人、都市压力感”的核心氛围
- 与现有像素 UI 视觉语言统一，不出现明显割裂
- 保证桌面端与移动端首页的标题、说明和按钮都清晰可读
- 让移动端用户可以在首页完成设置、读档、开新局，并直接进入游戏

## Chosen Direction

### Visual Style

- 画风：复古像素插画
- 基调：深蓝、灰紫、冷白屏幕光
- 情绪：克制、压抑、都市、加班感，不做成恐怖或赛博朋克海报

### Scene

- 主场景：深夜办公室
- 视角：从人物后方偏高一点看向工位
- 主体：一个背对镜头的打工人坐在工位前
- 环境：桌面设备、办公区、玻璃窗、窗外夜景与少量霓虹

### Desktop Composition

- 使用单张横版主视觉图
- 中间区域保持相对干净，优先给首页标题和按钮让位
- 画面细节集中在左右两侧与下半区
- 人物、工位和窗外夜景负责建立叙事氛围，但不抢占中心视觉锚点

### Mobile Strategy

- 移动端继续使用整屏背景图，不切换为纯色首页
- 可用性优先于沉浸感，前景改为更清晰的内容面板布局
- 手机端允许直接进入游戏，不再显示“请使用电脑访问”作为阻断
- 首页保留 `新游戏`、`读取存档`、`设置` 三个主入口

## Scene Breakdown

### Foreground

- 背对镜头的主角坐在电脑前，姿态略疲惫但不过度夸张
- 桌面可出现显示器、键盘、咖啡杯、便签、文件夹等职场元素
- 元素数量要克制，强调“真实工位”而不是“道具堆叠”

### Midground

- 可见数排昏暗工位或办公室内部结构
- 保留办公空间感，但避免加入太多角色或过强剧情动作
- 局部可有屏幕亮光、桌灯残光或空工位轮廓

### Background

- 通过玻璃窗看到城市夜景
- 允许少量霓虹、楼宇灯光、远处广告屏，增强都市感
- 不做过强赛博效果，不使用夸张高饱和 neon flooding

## Landing Page Structure

### Desktop Layout

- 桌面端继续保持偏中置的首页结构
- 游戏标题、副标题与主按钮悬浮在背景图中央安全区内
- 保留当前首页的像素风视觉语言，只替换背景层与必要布局细节

### Mobile Layout

- 手机首页不再只显示“请使用电脑访问”
- 背景层保持整屏展示，但叠加更强的暗色遮罩与渐变
- 前景为一块清晰的内容面板或半透明卡片区域
- 内容顺序为：标题、副标题、`新游戏`、`读取存档`、`设置`
- 三个主按钮采用纵向堆叠的大点击区域，适合手指操作
- 布局需要保留安全边距，兼容窄屏与刘海屏

## Mobile Background Strategy

- 首版优先使用与桌面端相同的主视觉背景图
- 移动端通过更重的遮罩、渐变与裁切来保证可读性
- 背景图焦点在窄屏裁切后仍需保留“人物 + 工位 + 深夜办公室”的识别度
- 如果后续验证发现同一张图在移动端裁切损失过大，允许补一张更适合手机构图的变体，但不作为第一阶段前置要求
- 背景图负责氛围表达，信息传达交给前景标题、文案与按钮

## Interaction Requirements

### Shared Entry Points

- 首页在桌面端与移动端都保留 `新游戏`、`读取存档`、`设置`
- `新游戏` 为主要行动入口，应具有最强视觉层级
- `读取存档` 和 `设置` 不进入二级菜单，不隐藏

### Mobile Behavior

- 点击 `新游戏` 后，移动端用户可以直接进入游戏
- `读取存档` 和 `设置` 在手机上应采用更适合小屏的弹层方式
- 弹层在移动端应接近全宽或接近全屏，避免桌面式小弹窗难点
- 按钮尺寸、间距和触控热区需要针对手指操作调整

### Fallback Behavior

- 若背景图未加载，首页仍可回退到现有深色背景或深色渐变
- 背景图是增强项，不应成为进入游戏的依赖项

## Asset Specification

- 首版至少制作 1 张桌面优先的横版首页背景图
- 图片中不出现 Logo、标题字或 UI 文案
- 输出应兼顾清晰度与体积，避免做成过重的大海报资源
- 第一版不包含动画，不拆分多层视差资源
- 资源在选型时要兼顾移动端裁切可用性

## Non-Goals

- 不在本轮承诺“整个游戏完整适配移动端”
- 不做多风格首页分支版本
- 不做动态粒子、CRT 扫描线、闪烁灯光等动画效果
- 不在本轮重做全部游戏界面与交互

## Acceptance Criteria

- 玩家第一眼能明确识别“深夜办公室里的打工人”主题
- 画面与现有像素风 UI 在风格上统一
- 桌面端首页中央标题和按钮区域清晰、稳定、可读
- 移动端首页可正常展示、可操作，并可直接进入游戏
- 移动端首页同时保留 `新游戏`、`读取存档`、`设置`
- 背景图能传达都市职场压力感，但不会喧宾夺主
- 整体更像游戏首页场景图，而不是独立宣传海报
