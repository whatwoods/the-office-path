# Homepage Background Design Spec

## Overview

为《打工之道》首页增加一张玩家向主视觉背景图，用于强化“深夜办公室里的打工人生”气质，同时保持当前像素风 UI 的可读性与统一性。

本次设计聚焦一张桌面端首页背景图，不扩展到动态背景、移动端专用版本或多套皮肤。目标不是做一张独立海报，而是做一张能和标题、按钮、现有配色稳定协作的首页场景图。

## Design Goal

- 第一眼就让玩家感受到“这是一个 AI 驱动的职场模拟器”
- 强化“深夜、办公室、打工人、都市压力感”的核心氛围
- 与现有像素 UI 视觉语言统一，不出现明显割裂
- 保证首页正中区域的标题、副标题和按钮始终清晰可读

## Chosen Direction

### Visual Style

- 画风：复古像素插画
- 基调：深蓝、灰紫、冷白屏幕光
- 情绪：克制、压抑、都市、加班感，不做成恐怖或赛博朋克海报

### Scene

- 主场景：深夜办公室
- 视角：从人物后方偏高一点看向工位
- 主体：一个背对镜头的打工人坐在工位前
- 环境：桌面设备、办公区、玻璃窗、窗外夜景与少量霓虹

### Composition

- 使用单张横版主视觉图
- 中间区域保持相对干净，优先给首页标题和按钮让位
- 画面细节集中在左右两侧与下半区
- 人物、工位和窗外夜景负责建立叙事氛围，但不抢占中心视觉锚点

## Scene Breakdown

### Foreground

- 背对镜头的主角坐在电脑前，姿态略疲惫但不过度夸张
- 桌面可出现显示器、键盘、咖啡杯、便签、文件夹等职场元素
- 元素数量要克制，强调“真实工位”而不是“道具堆叠”

### Midground

- 可见数排昏暗工位或办公室内部结构
- 保留办公空间感，但避免加入太多角色或过强剧情动作
- 局部可有屏幕亮光、桌灯残光或空工位轮廓

### Background

- 通过玻璃窗看到城市夜景
- 允许少量霓虹、楼宇灯光、远处广告屏，增强都市感
- 不做过强赛博效果，不使用夸张高饱和 neon flooding

## Integration Requirements

### Landing Page Usage

- 背景图服务于首页首屏，不内嵌任何文字
- 游戏标题“打工之道”和说明文案继续由前端渲染
- 背景图需要允许前端叠加轻微暗色遮罩，以稳定按钮和文字对比度

### Readability Rules

- 标题区域附近不能出现高亮、强对比或复杂主体
- 页面中部应以暗色过渡、弱细节、可控光感为主
- 背景图即使在不同显示器亮度下，也不能压掉按钮可见性

### Fallback Behavior

- 若背景图未加载，首页仍可退回当前纯色背景
- 背景图是增强项，不应成为进入游戏的依赖项

## Asset Specification

- 首版仅制作 1 张桌面端横版背景图
- 构图按宽屏首页使用场景设计
- 图片中不出现 Logo、标题字或 UI 文案
- 输出应兼顾清晰度与体积，避免做成过重的大海报资源
- 第一版不包含动画，不拆分多层视差资源

## Non-Goals

- 不做移动端首页专用插画
- 不做多风格分支版本
- 不做动态粒子、CRT 扫描线、闪烁灯光等动画效果
- 不在本轮处理整套首页重构，只聚焦背景图方向

## Acceptance Criteria

- 玩家第一眼能明确识别“深夜办公室里的打工人”主题
- 画面与现有像素风 UI 在风格上统一
- 首页中央标题和按钮区域清晰、稳定、可读
- 背景图能传达都市职场压力感，但不会喧宾夺主
- 整体更像游戏首页场景图，而不是独立宣传海报
